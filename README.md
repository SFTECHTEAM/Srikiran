# Srikiran - Meeting Intelligence
Telugu AI Meeting Recorder → Whisper → Llama3 → PDF

## How to get your .EXE and .APK (Super Easy!)

1. Upload ALL these files to GitHub
2. Go to **Actions** tab in your GitHub repo
3. Click **"Build Srikiran EXE + APK"**
4. Click **"Run workflow"** → **"Run workflow"**
5. Wait 10-15 minutes
6. Download your files from **Artifacts** section ✅

That's it!
